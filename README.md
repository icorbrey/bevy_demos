# Bevy Demos

This is a collection of demos written for [Bevy](https://bevyengine.org/).
Nothing in this repository should be considered production-ready.

| Demo          | Command                      |
|---------------|------------------------------|
| Block Breaker | `cargo run -- block-breaker` |
| Tilesets      | `cargo run -- tilesets`      |
